# TodoList Backend

## Docker-compose
Locate in docker-compose folder and run in terminal
```
docker-compose up -d
```

## Develop
Only in develop, it will insert default task states

Install
```
npm i
```

Start
```
npm run dev
```

## Test
For testing, use mongodb in memory, run:
```
npm run test
```
